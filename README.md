# arduino-uno-ssd1306-robot-eyes
Proyek robot eyes berbasis Arduino Uno yang menampilkan animasi mata pada layar OLED 0.96 inci menggunakan driver SSD1306. Kode dibagikan secara terbuka untuk tujuan pembelajaran, eksperimen embedded system, dan pengembangan robotika.
Tentang Proyek

Repository ini berisi proyek Robot Eyes menggunakan Arduino Uno dan OLED 0.96 inci (SSD1306). Proyek ini menampilkan animasi mata sederhana seperti membuka–menutup, melihat kiri–kanan, dan ekspresi dasar yang umum digunakan pada robot atau sistem interaksi visual.

Proyek ini ditujukan untuk:

Pembelajaran embedded system (sistem tertanam)

Eksperimen human–robot interaction (interaksi manusia–robot)

Referensi penggunaan OLED SSD1306 pada Arduino

Spesifikasi Teknis

Microcontroller: Arduino Uno (ATmega328P)

Display: OLED 0.96" (128×64)

Driver IC: SSD1306

Interface: I²C (SDA, SCL)

Bahasa Pemrograman: C/C++ (Arduino)

Koneksi Dasar (I²C)

VCC → 5V / 3.3V

GND → GND

SDA → A4

SCL → A5
